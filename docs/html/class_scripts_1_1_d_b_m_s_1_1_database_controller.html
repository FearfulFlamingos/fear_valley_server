Detailed Description

The Database controller is the main way to access the single database we created.

It is easily extensible, with method definitions for the four main operations:

Create(string)

Read(string)

Update(string)

Delete(string)

More complex queries exist as methods. These are purpose built for the code that requires them, since all database interactions are kept in the same file.
